Дата: 2026-05-19
Статус: завершено

## План
- Инструмент по образцу measure.py: курсор + F7 → HSV под курсором.

## Прогресс
- Создан pick_color.py: GetCursorPos (ctypes), патч 11x11 через mss,
  BGR→HSV, F7 — проба, F9 — выход.
- Несколько проб накапливаются: итоговый диапазон = min/max по всем,
  печатается готовой строкой HSVRange(...).
- Патч клампится в границы экрана. py_compile — OK.

## Решения и находки
- min/max по патчу даёт тугой диапазон; несколько проб расширяют его.
- Для красного Hue кольцевой — диапазон выйдет широким (отмечено в docstring).
